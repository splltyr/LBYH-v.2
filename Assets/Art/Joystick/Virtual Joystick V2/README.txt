====================================
 VIRTUAL JOYSTICK - README
====================================

Thank you for downloading **Virtual Joystick**!

This is a free and customizable virtual controller designed for mobile and PC game developers.
You can easily integrate it into your projects to handle player movement or touch-based input.

------------------------------------
📦 CONTENTS
------------------------------------
- /Assets/ or /Scripts/ folder → contains the joystick files
- preview.png → optional image showing how it looks
- LICENSE.txt → (optional) free use license

------------------------------------
⚙️ HOW TO USE
------------------------------------
1. Extract this .rar file using WinRAR or 7-Zip.
2. Open your game project (Godot, Unity, Unreal, etc.).
3. Import the joystick files into your project folder.
4. Assign the joystick UI to your player input system.
5. Adjust position, size, or transparency as needed.

------------------------------------
💡 TIPS
------------------------------------
- Works best with games that use touch or virtual input.
- You can customize visuals to match your UI style.
- Check your engine’s input settings for touch/mouse compatibility.
